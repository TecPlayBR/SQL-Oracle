# 📊 SQL Oracle - Estudos e Prática Profissional

Este repositório reúne scripts, consultas e materiais de estudo em **SQL utilizando Oracle**, desenvolvidos durante minha experiência profissional em uma empresa.

O objetivo é servir como:
- 📚 Material didático para aprendizado
- 💼 Portfólio técnico
- 🧠 Base de consulta para uso futuro

---

## 🚀 Conteúdo do Repositório

### 🗂️ Scripts SQL
Consultas e exemplos práticos desenvolvidos no dia a dia, incluindo:

- SELECTs básicos e avançados
- JOINs (INNER, LEFT, RIGHT)
- Subqueries
- Funções agregadas
- Manipulação de dados
- Estruturação de queries mais complexas

---

### 📘 Apostilas e Anotações
Material complementar com:

- Explicações conceituais

- Boas práticas

- Anotações pessoais durante o aprendizado
